## 📌 版本概述

v1.2.0 版本新增了完整的 TUI 设置模块，支持在交易界面中直接配置止盈止损参数、杠杆设置等，无需手动编辑配置文件。

## ✅ 已完成功能

### 1. TUI 设置面板

| 功能 | 说明 | 状态 |
|------|------|------|
| 进入/退出设置 | 按 `S` 键进入/退出设置面板 | ✅ |
| 字段切换 | 按 `↑↓` 切换配置字段 | ✅ |
| 数值调整 | 按 `←→` 调整数值或切换选项 | ✅ |
| 编辑模式 | 按 `Enter` 进入编辑模式，支持数字输入 | ✅ |
| 保存配置 | 按 `S` 保存并退出 | ✅ |
| 重置默认 | 按 `D` 重置为默认配置 | ✅ |
| 备份配置 | 按 `B` 备份当前配置 | ✅ |

### 2. 止盈设置（双模式）

| 模式 | 说明 | 示例 |
|------|------|------|
| 固定点数 | 开仓价 ± 固定点数 | +1.00 点 |
| 百分比 | 开仓价 × (1 ± 百分比%) | +0.36% |

### 3. 止损设置（双模式 + 双模式）

**触发价模式：**
| 模式 | 说明 | 示例 |
|------|------|------|
| 固定点数 | 开仓价 ± 固定点数 | -3.00 点 |
| 百分比 | 开仓价 × (1 ± 百分比%) | -0.50% |

**实际止损价模式：**
| 模式 | 说明 | API 参数 |
|------|------|---------|
| 同向价 1 | 触发后挂同向价 1 | `priceMatch='QUEUE'` |
| 自定义滑点 | 触发后挂指定滑点价格 | `price='2157.50'` |

### 4. 保底止损（最大损失比例）

基于用户配置的最大损失比例自动计算保底止损价格：

```
最大损失（USDT）= 开仓前保证金 × 最大损失比例
手续费 = 名义仓位价值 × 0.05%
实际可承受价格损失 = 最大损失 - 手续费
止损价 = 开仓价 ± 损失价差
```

**安全底线：** 如果计算的止损价比强平价 +1 更危险，则使用强平价 +1。

### 5. 双杠杆设置

| 杠杆 | 说明 | 影响 |
|------|------|------|
| API 杠杆 | 币安 API 设置的杠杆 | 影响最大可开名义价值 |
| 实际杠杆 | 实际仓位计算用的杠杆 | 控制风险 |

示例：
- API 杠杆：100x → 最大名义价值 = 保证金 × 100
- 实际杠杆：25x → 本单名义价值 = 保证金 × 25

### 6. 配置持久化

| 功能 | 说明 |
|------|------|
| 自动保存 | 修改配置后按 S 自动保存到 `config/runtime.json` |
| 配置备份 | 按 B 键备份配置到 `config/runtime.json.YYYYMMDD_HHMMSS` |
| 重置默认 | 按 D 键重置为默认配置 |
| 配置验证 | 保存时自动验证配置合理性 |

### 8. WebSocket 连接状态显示

### 9. Z 键市价全平

| 功能 | 说明 |

## 📋 配置参数说明

### config/runtime.json

```json
{
    "take_profit": {
        "mode": "fixed",           // fixed | percentage
        "points": 1.00,            // 固定点数模式
        "percent": 0.36            // 百分比模式
    },
    "stop_loss": {
        "trigger_mode": "fixed",   // fixed | percentage
        "trigger_points": 3.00,    // 触发价（固定点数）
        "trigger_percent": 0.50,   // 触发价（百分比）
        "limit_mode": "queue",     // queue | custom
        "limit_offset": 10.50      // 自定义滑点
    },
    "stop_market": {
        "max_loss_percent": 30.00  // 最大损失比例
    },
    "leverage": {
        "api": 100,                // API 杠杆
        "actual": 25               // 实际杠杆
    },
    "order_timeout_seconds": 2.00
}
```

### 配置验证规则

| 参数 | 最小值 | 最大值 | 说明 |
|------|--------|--------|------|
| 止盈点数 | 0.01 | 100 | 固定点数模式 |
| 止盈百分比 | 0.01 | 10 | 百分比模式 |
| 止损触发点数 | 0.01 | 100 | 固定点数模式 |
| 止损触发百分比 | 0.01 | 10 | 百分比模式 |
| 最大损失比例 | 0.1 | 80 | 百分比 |
| API 杠杆 | 1 | 125 | |
| 实际杠杆 | 1 | 125 | 不能大于 API 杠杆 |

## 🎮 操作说明

### 主交易界面

| 按键 | 功能 |
|------|------|
| `↑` | 做多（挂买单 @ 买一价） |
| `↓` | 做空（挂卖单 @ 卖一价） |
| `←` | 撤销挂单 |
| `→` | 提前平仓 |
| `S` | 进入设置面板 |
| `H` | 手动同步持仓 |
| `Z` | 市价全平（有持仓时） |
| `Q` | 退出程序 |

### 设置界面

| 按键 | 功能 |
|------|------|
| `↑↓` | 切换配置字段 |
| `←→` | 调整数值或切换选项 |
| `Enter` | 进入编辑模式（数值字段） |
| `S` | 保存并退出 |
| `D` | 重置为默认配置 |
| `B` | 备份当前配置 |
| `Esc` | 退出（放弃修改） |
