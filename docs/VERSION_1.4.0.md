## 📊 指标体系

### 1. 波动率指标（5 个）

| 指标 | 计算方式 | 更新频率 | 阈值告警 |
|------|---------|---------|---------|
| 1 分钟振幅 | (最高 - 最低) / 开盘 × 100% | 每秒 | < 0.03% 🟡 / > 0.3% 🔴 |
| 5 分钟振幅 | 近 5 根 K 线综合振幅 | 每分钟 | - |
| 1 小时平均振幅 | 近 60 根 K 线平均振幅 | 每分钟 | < 0.08% 🟡 |
| 振幅变化率 | 当前振幅 / 1 小时平均振幅 | 每秒 | < 0.8 减速 / > 1.2 加速 |
| ATR(14) | 14 周期平均真实波幅 | 每分钟 | - |

### 2. 流动性指标（3 个）

| 指标 | 计算方式 | 更新频率 | 阈值告警 |
|------|---------|---------|---------|
| 买卖价差 | ask[0] - bid[0] | 每秒 | - |
| 价差率 | 价差 / 中间价 × 100% | 每秒 | > 0.02% 🔴 |
| 订单簿深度 | bid 前 3 档 + ask 前 3 档总量 | 每秒 | < 500 ETH 🔴 |

### 3. 综合评分

**质量评分 (0-100 分)：**
- 波动率适中 (0.05%-0.15%)：+30 分
- 价差率低 (< 0.01%)：+30 分
- 订单簿深度高 (> 2000 ETH)：+20 分
- 振幅稳定 (变化率 0.8-1.2)：+20 分

**交易建议：**
- 🟢 评分 ≥ 70：适合交易
- 🟡 评分 40-69：观望
- 🔴 评分 < 40：暂停交易

## ⚠️ 已知限制

### 1. K 线数据订阅

当前版本**缺少 WebSocket K 线数据推送**，波动率指标无法实时更新。

**解决方案（待实现）：**
1. 使用币安 WebSocket K 线流：`wss://fstream.binance.com/ws/<symbol>@kline_1m`
2. 或者通过 REST API 定时轮询（不推荐，增加 API 压力）

**临时方案：**
- 使用历史 K 线初始化指标
- 等待后续版本完善 WebSocket 订阅

### 2. 关闭日志

Windows 关闭窗口时 shutdown 日志可能无法写入（操作系统强制终止进程）。

**建议：** 按 Q 键退出以确保日志完整。
