## 🆕 核心功能

### 1. 动态指标系统

#### 1.1 指标体系

**波动率指标（5 个）：**

| 指标 | 计算方式 | 频率 | 预警阈值 |
|------|---------|------|---------|
| 1 分钟振幅 | (最高 - 最低) / 最新价 × 100% | 每秒 | < 0.03% 🟥 / > 0.3% 🟩 |
| 5 分钟振幅 | 近 5 根 K 线综合振幅 | 每分钟 | - |
| 1 小时平均振幅 | 近 60 根 K 线平均振幅 | 每分钟 | < 0.08% 🟥 |
| 变化率 | 当前价相对 1 小时前变化 | 每秒 | < 0.5% 🟥 / > 2% 🟩 |
| ATR(14) | 14 周期平均真实波幅 | 每分钟 | - |

**流动性指标（3 个）：**

| 指标 | 计算方式 | 频率 | 预警阈值 |
|------|---------|------|---------|
| 买卖价差 | ask - bid | 每秒 | < 0.01 USDC 🟩 |
| 价差率 | 价差 / 中间价 × 100% | 每秒 | < 0.05% 🟩 / > 0.2% 🟥 |
| 市场深度 | 前 10 档挂单总量 | 每秒 | > 50 ETH 🟩 |
| 深度不平衡 | (买量 - 卖量) / 总量 | 每秒 | -0.2 ~ 0.2 🟩 |

#### 1.2 综合评分

**分类评分权重：**
- 波动率 (Volatility)：40%
- 流动性 (Liquidity)：35%
- 动量 (Momentum)：25%

**信号等级：**
| 分数区间 | 信号 | 含义 | 建议 |
|---------|------|------|------|
| 60-100 | 🟢 适合 | 市场条件良好 | 可正常交易 |
| 30-59 | 🟡 观望 | 部分指标不理想 | 谨慎操作 |
| 0-29 | 🔴 暂停 | 市场条件不佳 | 暂停交易 |

#### 2.3 性能优化

- **批量写入** - 每 10 条记录批量写入一次
- **异步支持** - 不阻塞主线程
- **自动分文件** - 单文件上限 100MB 或 1000 条记录
- **压缩存储** - JSONL 格式，便于流式读取

## 📈 数据使用

### 查询指标数据

```python
import json
from pathlib import Path

metrics_file = Path("data/metrics/ETHUSDC_2026-03-28.jsonl")

with open(metrics_file, 'r', encoding='utf-8') as f:
    for line in f:
        record = json.loads(line)
        print(f"{record['datetime']}: {record['score_data']['total_score']}分 - {record['score_data']['recommendation']}")
```

### 统计分析

```python
# 计算平均评分
scores = []
with open(metrics_file, 'r', encoding='utf-8') as f:
    for line in f:
        record = json.loads(line)
        scores.append(record['score_data']['total_score'])

print(f"平均评分：{sum(scores)/len(scores):.1f}分")
print(f"最高评分：{max(scores)}分")
print(f"最低评分：{min(scores)}分")
```

## ⚠️ 注意事项

1. **磁盘空间** - 实时数据记录会占用较多磁盘空间（约 2MB/小时）
2. **性能影响** - 批量写入已优化，对主线程影响 < 1%
3. **数据备份** - 建议定期备份 `data/` 目录
4. **文件分割** - 单文件超过 100MB 自动分割
