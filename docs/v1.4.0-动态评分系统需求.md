## 📊 一、系统概述

### 1.1 核心目标

基于历史数据百分位数，动态评估盘面质量，给出 0-100 分综合评分 + 交易建议（适合/观望/暂停）。

### 1.2 设计原则

1. **自适应**：阈值自动跟随市场变化，不用固定值
2. **健壮性**：考虑样本量、分布形态、异常值
3. **可解释**：每个分数都有历史百分位依据
4. **程序化**：完全自动化，无需人工调整

## 📐 二、评分结构

### 2.1 综合评分公式

```
综合评分 = 波动率评分 × 30% + 流动性评分 × 40% + 动量评分 × 30%
```

### 2.2 指标明细

| 分类 | 权重 | 指标 | 计算方式 | 越高越好？ |
|------|------|------|---------|-----------|
| **波动率** | 30% | 1 分钟振幅 | (High-Low)/Open × 100% | ❌ 越低越好 |
| | | 3 小时平均振幅 | 180 根 1 分钟 K 线振幅均值 | ❌ 越低越好 |
| **流动性** | 40% | 价差率 | (Ask1-Bid1)/MidPrice × 100% | ❌ 越低越好 |
| | | 表层深度 (20 档) | Bid1-20 + Ask1-20 总量 | ✅ 越高越好 |
| | | 中层深度 (100 档) | Bid21-100 + Ask21-100 总量 | ✅ 越高越好 |
| | | 深层深度 (1000 档) | Bid101-1000 + Ask101-1000 总量 | ✅ 越高越好 |
| | | 深度不平衡 | (Bid 总量 - Ask 总量) / 总量 | ❌ 越接近 0 越好 |
| **动量** | 30% | ATR(14) | 14 周期平均真实波幅 | ✅ 越高越好 |
| | | 成交量趋势 | 当前量 / 过去 100 根均量 | ✅ 越高越好 |

## 🔄 三、动态评分算法

### 3.1 单个指标评分函数

```python
def percentile_score(current_value, historical_data, higher_is_better=True):
    """
    基于历史百分位数评分（0-100 分）
    
    参数：
    - current_value: 当前值
    - historical_data: 历史数据列表
    - higher_is_better: True=越高越好，False=越低越好
    
    返回：0-100 分
    """
    if not historical_data:
        return 50  # 默认中等分数
    
    # 计算当前值的百分位排名
    sorted_data = sorted(historical_data)
    count_below = sum(1 for x in sorted_data if x <= current_value)
    percentile_rank = count_below / len(sorted_data) * 100
    
    if higher_is_better:
        score = percentile_rank  # 越高分数越高
    else:
        score = 100 - percentile_rank  # 越低分数越高
    
    return max(0, min(100, score))
```

### 3.2 样本量置信度调整

```python
def confidence_weight(sample_count, target_count=100):
    """
    根据样本量计算置信度权重
    - < 30 个：统计学上不足以计算百分位数
    - 30-100 个：可参考但需打折
    - > 100 个：可靠
    """
    if sample_count < 30:
        return sample_count / 30 * 0.5  # 最多 50% 权重
    elif sample_count < 100:
        return 0.5 + (sample_count - 30) / 70 * 0.5  # 50%-100%
    else:
        return 1.0
```

### 3.3 分布形态检验

```python
from scipy import stats

def check_distribution(data):
    """
    检验数据分布形态
    返回：分布类型 + 置信度
    """
    if len(data) < 30:
        return 'unknown', 0
    
    # 正态性检验（Shapiro-Wilk）
    stat, p_value = stats.shapiro(data)
    
    if p_value > 0.05:
        return 'normal', p_value  # 正态分布
    else:
        return 'non-normal', p_value  # 偏态分布
```

**评分调整：**
- **正态分布**：用 Z-score 标准化后评分
- **偏态分布**：用百分位数评分

### 3.4 异常值处理（IQR 方法）

```python
def remove_outliers_iqr(data):
    """
    用 IQR 方法去除异常值
    """
    q1 = percentile(data, 25)
    q3 = percentile(data, 75)
    iqr = q3 - q1
    
    lower_bound = q1 - 1.5 * iqr
    upper_bound = q3 + 1.5 * iqr
    
    return [x for x in data if lower_bound <= x <= upper_bound]
```

## 📊 四、流动性分层计算

### 4.1 分层方案

| 层级 | 档位范围 | 权重 | 意义 |
|------|---------|------|------|
| 表层 | Bid1-20 / Ask1-20 | 50% | 高频交易，反映短期压力 |
| 中层 | Bid21-100 / Ask21-100 | 30% | 机构订单，反映中期趋势 |
| 深层 | Bid101-1000 / Ask101-1000 | 20% | 大户订单，反映长期支撑 |

### 4.2 计算函数

```python
def analyze_depth_layers(orderbook):
    """分层分析深度"""
    bids = orderbook['bids']
    asks = orderbook['asks']
    
    layers = {
        'surface': (0, 20),
        'middle': (20, 100),
        'deep': (100, 1000)
    }
    
    results = {}
    for layer_name, (start, end) in layers.items():
        bid_depth = sum(float(b[1]) for b in bids[start:end])
        ask_depth = sum(float(a[1]) for a in asks[start:end])
        total_depth = bid_depth + ask_depth
        imbalance = (bid_depth - ask_depth) / total_depth if total_depth > 0 else 0
        
        results[layer_name] = {
            'bid_depth': bid_depth,
            'ask_depth': ask_depth,
            'total_depth': total_depth,
            'imbalance': imbalance
        }
    
    return results
```

### 4.3 流动性评分（分层加权）

```python
def calc_liquidity_score(current_data, historical_data):
    """流动性评分（分层加权）"""
    surface_score = percentile_score(
        current_data['depth_surface'],
        historical_data['depth_surface'],
        higher_is_better=True
    )
    middle_score = percentile_score(
        current_data['depth_middle'],
        historical_data['depth_middle'],
        higher_is_better=True
    )
    deep_score = percentile_score(
        current_data['depth_deep'],
        historical_data['depth_deep'],
        higher_is_better=True
    )
    
    # 加权平均
    liquidity_score = (
        surface_score * 0.50 +
        middle_score * 0.30 +
        deep_score * 0.20
    )
    
    return liquidity_score
```

### 5.2 scorer.py 核心类

```python
class DynamicScorer:
    """动态评分器 - 自动维护历史数据窗口"""
    
    def __init__(self, window_days=14):
        self.window_days = window_days
        self.historical_data = {
            '1min_amplitude': [],
            '3h_avg_amplitude': [],
            'spread_rate': [],
            'depth_surface': [],
            'depth_middle': [],
            'depth_deep': [],
            'depth_imbalance': [],
            'atr_14': [],
            'volume_trend': [],
        }
    
    def update_history(self, new_data):
        """添加新数据，维护滚动窗口"""
        for key, value in new_data.items():
            if key in self.historical_data:
                self.historical_data[key].append({
                    'value': value,
                    'timestamp': datetime.now().isoformat()
                })
        
        # 清理过期数据（超过 window_days）
        cutoff = datetime.now() - timedelta(days=self.window_days)
        for key in self.historical_data:
            self.historical_data[key] = [
                d for d in self.historical_data[key]
                if datetime.fromisoformat(d['timestamp']) > cutoff
            ]
    
    def score(self, current_data):
        """计算当前盘面评分"""
        # 提取历史值列表
        history = {
            key: [d['value'] for d in values]
            for key, values in self.historical_data.items()
        }
        
        # 1. 波动率评分（30%）
        vol_score = self._calc_volatility_score(current_data, history)
        
        # 2. 流动性评分（40%）
        liq_score = self._calc_liquidity_score(current_data, history)
        
        # 3. 动量评分（30%）
        mom_score = self._calc_momentum_score(current_data, history)
        
        # 4. 综合评分
        total_score = (
            vol_score * 0.30 +
            liq_score * 0.40 +
            mom_score * 0.30
        )
        
        # 5. 交易建议
        if total_score >= 75:
            recommendation = "适合交易"
        elif total_score >= 50:
            recommendation = "观望"
        else:
            recommendation = "暂停交易"
        
        return {
            'total_score': round(total_score, 1),
            'category_scores': {
                'volatility': round(vol_score, 1),
                'liquidity': round(liq_score, 1),
                'momentum': round(mom_score, 1),
            },
            'recommendation': recommendation,
            'timestamp': datetime.now().isoformat()
        }
```
