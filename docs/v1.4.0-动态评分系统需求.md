## 📊 一、系统概述

### 1.1 核心目标

基于历史数据百分位数，动态评估盘面质量，给出 0-100 分综合评分 + 交易建议（适合/观望/暂停）。

### 1.2 设计原则

1. **自适应**：阈值自动跟随市场变化，不用固定值
2. **健壮性**：考虑样本量、分布形态、异常值
3. **可解释**：每个分数都有历史百分位依据
4. **程序化**：完全自动化，无需人工调整

---

## 📐 二、评分结构

### 2.1 综合评分公式

```
综合评分 = 波动率评分 × 30% + 流动性评分 × 40% + 动量评分 × 30%
```

### 2.2 指标明细

| 分类 | 权重 | 指标 | 计算方式 | 越高越好？ | 必需？ |
|------|------|------|---------|-----------|--------|
| **波动率** | 30% | 1 分钟振幅 | (High-Low)/Open × 100% | ❌ 越低越好 | ✅ 必需 |
| | | **60 分钟平均振幅** | 60 根 1 分钟 K 线振幅均值 | ❌ 越低越好 | ✅ 必需 |
| **流动性** | 40% | 价差率 | (Ask1-Bid1)/MidPrice × 100% | ❌ 越低越好 | ✅ 必需 |
| | | 表层深度 (20 档) | Bid1-20 + Ask1-20 总量 | ✅ 越高越好 | ✅ 必需 |

## ⚠️ 三、关键调整（v1.1 更新）

### 3.1 K 线振幅计算 - 剔除当前 K 线

**重要：** 所有振幅统计都不包含当前正在形成的 K 线，只用已完成的 K 线。

```python
def analyze_volatility(klines, lookback=60):
    """
    分析波动率 - 只用已完成的 K 线
    
    ⚠️ 关键：排除当前正在形成的 K 线
    """
    if len(klines) < lookback + 1:
        return None
    
    # 排除当前 K 线，只用已完成的
    completed_klines = klines[:-1]
    
    # 1 分钟振幅 = 上一根已完成 K 线的振幅
    last_completed = completed_klines[-1]
    amplitude_1min = (last_completed[2] - last_completed[3]) / last_completed[1] * 100
    
    # 60 分钟平均振幅 = 过去 60 根已完成 K 线的振幅均值
    recent_60 = completed_klines[-lookback:]
    amplitudes = [(k[2] - k[3]) / k[1] * 100 for k in recent_60]
    amplitude_60min = sum(amplitudes) / len(amplitudes)
    
    return {
        '1min_amplitude': amplitude_1min,
        '60min_avg_amplitude': amplitude_60min,
    }
```

**原因：**
- 当前 K 线还未完成，数据不稳定
- 交易决策应基于已确认的市场状态

```python
def aggregate_orderbook(orderbook, price_step=0.1):
    """
    聚合订单簿到指定价格精度
    
    用于补充深层流动性分析
    """
    aggregated_bids = {}
    aggregated_asks = {}
    
    for price, volume in orderbook['bids']:
        price = float(price)
        volume = float(volume)
        # 向下取整到最近的 price_step
        agg_price = math.floor(price / price_step) * price_step
        aggregated_bids[agg_price] = aggregated_bids.get(agg_price, 0) + volume
    
    for price, volume in orderbook['asks']:
        price = float(price)
        volume = float(volume)
        agg_price = math.ceil(price / price_step) * price_step
        aggregated_asks[agg_price] = aggregated_asks.get(agg_price, 0) + volume
    
    # 转换为列表
    bids_list = sorted(aggregated_bids.items(), key=lambda x: x[0], reverse=True)
    asks_list = sorted(aggregated_asks.items(), key=lambda x: x[0])
    
    return bids_list, asks_list

# 使用示例
# 聚合到 0.1 USDC 档位，可覆盖更深的价格范围
agg_bids, agg_asks = aggregate_orderbook(orderbook, price_step=0.1)
deep_depth = sum(b[1] for b in agg_bids[:50]) + sum(a[1] for a in agg_asks[:50])
```

## 🔄 四、动态评分算法

### 4.1 单个指标评分函数

```python
def percentile_score(current_value, historical_data, higher_is_better=True):
    """
    基于历史百分位数评分（0-100 分）
    
    参数：
    - current_value: 当前值
    - historical_data: 历史数据列表
    - higher_is_better: True=越高越好，False=越低越好
    
    返回：0-100 分
    """
    if not historical_data:
        return 50  # 默认中等分数
    
    # 计算当前值的百分位排名
    sorted_data = sorted(historical_data)
    count_below = sum(1 for x in sorted_data if x <= current_value)
    percentile_rank = count_below / len(sorted_data) * 100
    
    if higher_is_better:
        score = percentile_rank  # 越高分数越高
    else:
        score = 100 - percentile_rank  # 越低分数越高
    
    return max(0, min(100, score))
```

### 4.2 样本量置信度调整

```python
def confidence_weight(sample_count, target_count=100):
    """
    根据样本量计算置信度权重
    - < 30 个：统计学上不足以计算百分位数
    - 30-100 个：可参考但需打折
    - > 100 个：可靠
    """
    if sample_count < 30:
        return sample_count / 30 * 0.5  # 最多 50% 权重
    elif sample_count < 100:
        return 0.5 + (sample_count - 30) / 70 * 0.5  # 50%-100%
    else:
        return 1.0
```

### 4.3 分布形态检验

```python
from scipy import stats

def check_distribution(data):
    """
    检验数据分布形态
    返回：分布类型 + 置信度
    """
    if len(data) < 30:
        return 'unknown', 0
    
    # 正态性检验（Shapiro-Wilk）
    stat, p_value = stats.shapiro(data)
    
    if p_value > 0.05:
        return 'normal', p_value  # 正态分布
    else:
        return 'non-normal', p_value  # 偏态分布
```

**评分调整：**
- **正态分布**：用 Z-score 标准化后评分
- **偏态分布**：用百分位数评分

### 4.4 异常值处理（IQR 方法）

```python
def remove_outliers_iqr(data):
    """
    用 IQR 方法去除异常值
    """
    q1 = percentile(data, 25)
    q3 = percentile(data, 75)
    iqr = q3 - q1
    
    lower_bound = q1 - 1.5 * iqr
    upper_bound = q3 + 1.5 * iqr
    
    return [x for x in data if lower_bound <= x <= upper_bound]
```

### 5.1 分层方案（调整后）

| 层级 | 档位范围 | 权重 | 意义 |
|------|---------|------|------|
| 表层 | Bid1-20 / Ask1-20 | 50% | 高频交易，反映短期压力 |
| 中层 | Bid21-200 / Ask21-200 | 40% | 机构订单，反映中期趋势 |
| 聚合层 | 0.1 USDC 聚合前 50 档 | 10% | 补充深层流动性 |

### 5.2 计算函数（更新版）

```python
def analyze_depth_layers(orderbook):
    """分层分析深度（200 档 + 聚合分析）"""
    bids = orderbook['bids']
    asks = orderbook['asks']
    
    # 表层（20 档）
    bid_depth_surface = sum(float(b[1]) for b in bids[:20])
    ask_depth_surface = sum(float(a[1]) for a in asks[:20])
    total_surface = bid_depth_surface + ask_depth_surface
    imbalance_surface = (bid_depth_surface - ask_depth_surface) / total_surface if total_surface > 0 else 0
    
    # 中层（21-200 档）
    bid_depth_middle = sum(float(b[1]) for b in bids[20:200])
    ask_depth_middle = sum(float(a[1]) for a in asks[20:200])
    total_middle = bid_depth_middle + ask_depth_middle
    imbalance_middle = (bid_depth_middle - ask_depth_middle) / total_middle if total_middle > 0 else 0
    
    # 聚合层（0.1 USDC 聚合，模拟深层）
    agg_bids, agg_asks = aggregate_orderbook(orderbook, price_step=0.1)
    bid_depth_agg = sum(float(b[1]) for b in agg_bids[:50])
    ask_depth_agg = sum(float(a[1]) for a in agg_asks[:50])
    total_agg = bid_depth_agg + ask_depth_agg
    imbalance_agg = (bid_depth_agg - ask_depth_agg) / total_agg if total_agg > 0 else 0
    
    return {
        'surface': {
            'bid_depth': bid_depth_surface,
            'ask_depth': ask_depth_surface,
            'total_depth': total_surface,
            'imbalance': imbalance_surface
        },
        'middle': {
            'bid_depth': bid_depth_middle,
            'ask_depth': ask_depth_middle,
            'total_depth': total_middle,
            'imbalance': imbalance_middle
        },
        'aggregated': {
            'bid_depth': bid_depth_agg,
            'ask_depth': ask_depth_agg,
            'total_depth': total_agg,
            'imbalance': imbalance_agg
        }
    }
```

### 5.3 流动性评分（更新版）

```python
def calc_liquidity_score(current_data, historical_data):
    """流动性评分（分层加权 - 200 档版本）"""
    surface_score = percentile_score(
        current_data['depth_surface'],
        historical_data['depth_surface'],
        higher_is_better=True
    )
    middle_score = percentile_score(
        current_data['depth_middle'],
        historical_data['depth_middle'],
        higher_is_better=True
    )
    agg_score = percentile_score(
        current_data['depth_aggregated'],
        historical_data['depth_aggregated'],
        higher_is_better=True
    )
    
    # 加权平均（50% + 40% + 10%）
    liquidity_score = (
        surface_score * 0.50 +
        middle_score * 0.40 +
        agg_score * 0.10
    )
    
    return liquidity_score
```

### 6.2 scorer.py 核心类（更新版）

```python
class DynamicScorer:
    """动态评分器 - 自动维护历史数据窗口"""
    
    def __init__(self, window_days=14):
        self.window_days = window_days
        self.historical_data = {
            # 波动率指标
            '1min_amplitude': [],       # 1 分钟振幅（已完成 K 线）
            '60min_avg_amplitude': [],  # 60 分钟平均振幅（已完成 K 线）
            
            # 流动性指标
            'spread_rate': [],
            'depth_surface': [],        # 表层 20 档
            'depth_middle': [],         # 中层 180 档
            'depth_aggregated': [],     # 聚合层
            'depth_imbalance': [],
            
            # 动量指标
            'atr_14': [],
            # 'volume_trend': [],  # v1.5 迭代添加
        }
    
    def update_history(self, new_data):
        """添加新数据，维护滚动窗口"""
        for key, value in new_data.items():
            if key in self.historical_data:
                self.historical_data[key].append({
                    'value': value,
                    'timestamp': datetime.now().isoformat()
                })
        
        # 清理过期数据（超过 window_days）
        cutoff = datetime.now() - timedelta(days=self.window_days)
        for key in self.historical_data:
            self.historical_data[key] = [
                d for d in self.historical_data[key]
                if datetime.fromisoformat(d['timestamp']) > cutoff
            ]
    
    def score(self, current_data):
        """计算当前盘面评分"""
        # 提取历史值列表
        history = {
            key: [d['value'] for d in values]
            for key, values in self.historical_data.items()
        }
        
        # 1. 波动率评分（30%）
        vol_score = self._calc_volatility_score(current_data, history)
        
        # 2. 流动性评分（40%）
        liq_score = self._calc_liquidity_score(current_data, history)
        
        # 3. 动量评分（30%）
        mom_score = self._calc_momentum_score(current_data, history)
        
        # 4. 综合评分
        total_score = (
            vol_score * 0.30 +
            liq_score * 0.40 +
            mom_score * 0.30
        )
        
        # 5. 交易建议
        if total_score >= 75:
            recommendation = "适合交易"
        elif total_score >= 50:
            recommendation = "观望"
        else:
            recommendation = "暂停交易"
        
        return {
            'total_score': round(total_score, 1),
            'category_scores': {
                'volatility': round(vol_score, 1),
                'liquidity': round(liq_score, 1),
                'momentum': round(mom_score, 1),
            },
            'recommendation': recommendation,
            'timestamp': datetime.now().isoformat()
        }
```

### 9.2 深度不平衡：用哪一层的深度？

**答：用表层（20 档）+ 中层（200 档）加权，深层不单独计算。**

```python
# 表层不平衡（权重 60%）
imbalance_surface = (bid_20 - ask_20) / (bid_20 + ask_20)

# 中层不平衡（权重 40%）
imbalance_middle = (bid_200 - ask_200) / (bid_200 + ask_200)

# 综合不平衡
imbalance_total = imbalance_surface * 0.60 + imbalance_middle * 0.40

# 评分：越接近 0 越好
imbalance_score = 100 - min(100, abs(imbalance_total) * 200)
```

**原因：**
- 表层反映短期买卖压力（权重高）
- 中层反映中期趋势（权重低）
- 深层用聚合分析补充流动性，不单独计算不平衡
