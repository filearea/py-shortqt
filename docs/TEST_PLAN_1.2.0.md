## 📋 测试前准备

### 1. 切换到开发分支

```bash
cd D:\Project\py-shortqt
git checkout develop-1.2.0
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置账户

确保 `config/accounts.json` 已配置：

```json
{
    "accounts": [
        {
            "name": "主账号",
            "api_key": "YOUR_API_KEY",
            "api_secret": "YOUR_API_SECRET",
            "testnet": true
        }
    ]
}
```

### 4. 配置文件

`config/runtime.json` 会自动创建（首次运行），或使用默认配置。

### 4. 配置计算测试

#### 4.1 止盈计算

**配置：**
- 止盈模式：固定点数，1.00 点
- 假设开仓价：2150.00

**预期：**
- [ ] 止盈价 = 2151.00

**配置：**
- 止盈模式：百分比，0.36%
- 假设开仓价：2150.00

**预期：**
- [ ] 止盈价 = 2157.74

#### 4.2 止损计算（多单）

**配置：**
- 止损触发模式：固定点数，3.00 点
- 实际止损价模式：同向价 1
- 假设开仓价：2150.00

**预期：**
- [ ] 触发价 = 2147.00
- [ ] 实际止损价 = 2146.00（触发后挂单）

**配置：**
- 止损触发模式：固定点数，3.00 点
- 实际止损价模式：自定义滑点，10.50 点
- 假设开仓价：2150.00

**预期：**
- [ ] 触发价 = 2147.00
- [ ] 实际止损价 = 2157.50（2147 + 10.50）⚠️ **重点验证**

#### 4.3 保底止损计算

**配置：**
- 最大损失比例：30%
- 假设开仓价：2150.00
- 持仓：0.407 ETH
- 开仓前保证金：35.00 U
- 强平价：2060.00

**计算：**
1. 名义仓位价值 = 2150 × 0.407 = 875.05 U
2. 手续费 = 875.05 × 0.0005 = 0.44 U
3. 最大损失 = 35 × 30% = 10.50 U
4. 实际价格损失 = 10.50 - 0.44 = 10.06 U
5. 损失价差 = 10.06 / 0.407 = 24.72 U
6. 计算止损价 = 2150 - 24.72 = 2125.28
7. 强平价 +1 = 2061
8. 最终止损价 = max(2125.28, 2061) = 2125.28

**预期：**
- [ ] 保底止损价 = 2125.28

#### 4.4 盈亏比验证

**配置：**
- 止盈价：2157.74
- 止损触发价：2147.00

**计算：**
- 止盈幅度 = 7.74
- 止损幅度 = 3.00
- 盈亏比 = 7.74 / 3.00 = 2.58:1

**预期：**
- [ ] 显示盈亏比 ≥ 1.5:1 ✓
- [ ] 如果 < 1.5:1，显示 ⚠️ 警告
