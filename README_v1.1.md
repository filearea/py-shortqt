## 📝 配置文件

### 多账号配置 (`config/accounts.json`)

```json
{
    "accounts": [
        {
            "name": "主账号",
            "api_key": "your_api_key",
            "api_secret": "your_api_secret",
            "testnet": true
        },
        {
            "name": "备用账号",
            "api_key": "your_api_key_2",
            "api_secret": "your_api_secret_2",
            "testnet": false
        }
    ]
}
```

⚠️ **重要：** 实盘模式会显示本机外网 IP，请确保在币安 API 设置中添加此 IP 到白名单！
