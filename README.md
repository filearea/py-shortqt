## 🆕 v1.7.0 新功能

### 24 小时交易统计

自动追踪完整交易轮次（开仓+平仓），实时统计：
- 开仓/平仓次数
- 胜率（盈利轮次占比）
- 盈亏比（总盈利 / 总亏损）
- 累计盈亏

### 历史持仓面板

TUI 实时展示最近 10 条历史持仓，每条卡片显示：
- 方向（LONG/SHORT）+ 状态（未平仓/完全平仓/部分平仓）
- 开仓价、平仓价、时间
- 手续费、资费（资金费率）、净利

### 音效提醒

成交和关键事件自动播放提示音（Windows `ding.wav`）：
- 开仓成交：响 1 声
- 平仓/止损/止盈：响 3 声
- 持仓超时保本：响 4 声
- 系统设置 → 系统设置 tab 可关闭音效

### 订单簿自适应

- 买卖档数量一致，最新价永远居中
- 最多各 20 档，窗口大小变化时自动调整

### 买卖深度固定宽度

- 深度值固定 4 位整数 + 2 位小数（如 `0015.50 ETH`）
- 两边深度接近时显示灰色"势均力敌"

### ATR 波动率指标

TUI 指标区新增 ATR(14) 值和波动率百分比显示

### 部分成交处理

挂单部分成交时自动处理剩余数量，支持超时撤单容错

### 代码质量

- 全部 `print()` 替换为 `log_manager` 日志系统
- 余额同步节流优化（有挂单 5s / 无挂单 15s / 历史 30s）

## 🆕 v1.6.0 新功能

### 剥头皮评分系统（Scalping Scorer）

全新设计的三维评分体系，专为超短趋势剥头皮策略优化：

**评分维度：**
- **趋势强度（40%）**：K 线连续性 + Tick 级震荡检测
- **波动率匹配（30%）**：当前振幅 vs 止盈目标距离
- **深度充足度（30%）**：反向挂单量 vs 开仓所需 ETH

**输出结果：**
- 综合分数（0-100）
- 预计方向（看多/看空/无方向）
- 置信度百分比
- 分类评分（趋势 / 波动 / 深度）

### Tick 级震荡检测（Tick Tracker）

- 30 秒滑动窗口记录 bookTicker 价格
- 检测秒级反转次数（如 2010→2012→2010→2012）
- 计算窗口内最大振幅和动量方向
- 辅助评分系统识别剧烈行情

### K 线连续性分析（K-line Streak）

- 统计最近 N 根已收盘 K 线的连续涨跌方向
- 支持最多 10 根回看
- 识别单边趋势强度

### 动态参数挂钩

- 评分参数与止盈止损点数、账户余额、杠杆倍数联动
- 开仓所需 ETH = (余额 × 杠杆) / 当前价格
- 止盈目标距离（百分比）随价格自动调整

### WebSocket 组合流修复

- 修复多流订阅 URL 格式，kline 数据可靠接收
- 组合流消息正确解包 `{"stream":"...", "data":{...}}`

### ATR 缓存修复

- 修复 `deque(maxlen)` 导致 ATR 缓存永不刷新的 bug
- 改用 K 线时间戳判断缓存有效性
- 确保每分钟 K 线收盘时 ATR(14) 正确更新

## 🆕 v1.5.0 新功能

### 浮盈移动止损（Trailing Stop）

盈利时自动上移止损，逐级锁定利润，避免过早平仓。

**核心特性：**
- ✅ 开仓价 → 目标止盈价之间均分 N 格
- ✅ 价格每突破一格，止损自动上移一格
- ✅ 支持多单/空单对称逻辑
- ✅ 超过 8 格时自动滚动策略（币安条件单上限）
- ✅ 订单类型：Algo Order `STOP` + `priceMatch='QUEUE'`（Maker 成交）

**配置示例：**
```json
{
  "trailing_stop": {
    "enabled": true,
    "grid_count": 5
  }
}
```

**算法示意（多单，开仓价 2100，止盈 2115，5 格）：**
```
2115  ← 目标止盈
2112  ← 第4格（触发后止损设在 2109）
2109  ← 第3格（触发后止损设在 2106）
2106  ← 第2格（触发后止损设在 2103）
2103  ← 第1格（触发后止损设在 2100 保本）
2100  ← 开仓价
```

---

### 浮亏保护止损（Loss Protection）

开仓一段时间后仍浮亏，自动将止盈下移到开仓价，确保反弹时保本出场。

**核心特性：**
- ✅ 可配置触发时间（1-60 分钟）
- ✅ 仅在浮亏状态下触发
- ✅ 止盈单修改为开仓价（Maker 保本）
- ✅ 触发后不再重复操作

**配置示例：**
```json
{
  "loss_protection": {
    "enabled": true,
    "trigger_minutes": 5
  }
}
```

- 移动止损：开关 + 格数调节（3-10）
- 浮亏保护：开关 + 触发时间调节（1-60 分钟）

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置 API Key

```bash
cp config/accounts.json.example config/accounts.json
```

编辑 `config/accounts.json`，填入币安 API Key。**重要：** 在币安官网添加服务器 IP 到 API 白名单。

### 3. 运行

```bash
# Windows 双击启动（推荐）
双击 启动.bat

# 命令行
python launcher.py      # 启动器（选择模式/账户）
python src/main_live.py  # 直接实盘
python src/main_sim.py   # 直接模拟
```

## 📊 动态指标系统

实时计算多项关键市场指标，配合剥头皮评分系统提供交易建议：

| 类别 | 指标 | 说明 | 更新频率 |
|------|------|------|----------|
| 波动率 | 上根K线振幅 | 上一根已收盘 1min K 线振幅 | 每分钟 |
| | 5 分钟振幅 | 近 5 根 K 线综合振幅 | 每分钟 |
| | 1 小时振幅 | 近 60 根 K 线最高-最低 | 每分钟 |
| | 变化率 | 当前振幅 / ATR(60) | 每分钟 |
| | ATR(14) | 14 周期平均真实波幅 | 每分钟 |
| 流动性 | 买卖价差 | ask1 - bid1 | 100ms |
| | 价差率 | 价差 / 中间价 | 100ms |
| | 买盘深度 | 前 10 档买单总量 | 100ms |
| | 卖盘深度 | 前 10 档卖单总量 | 100ms |
| 趋势 | K 线连续性 | 连续同方向 K 线数量 | 每分钟 |
| | Tick 反转 | 30 秒窗口内价格反转次数 | 实时 |

**剥头皮评分（v1.6.0）：**
- 趋势强度 40% + 波动率匹配 30% + 深度充足度 30%
- 输出：综合分数 + 预计方向 + 置信度 + 分类评分
